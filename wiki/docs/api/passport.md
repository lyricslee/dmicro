# passport模块

## 发送短信验证码

接口描述: 发送短信验证码

请求路径:  /passport/Sms

请求方法:  POST

请求参数 : 参数类型 body
| 参数     | 值                  | 类型   | 说明          | 必须   | 
|:---------|:--------------------|:-------|:--------------|:-------|
| mobile   | 13805918888         | String | 手机号        | true   |

示例：
```json
{
  "mobile": "13805918888"
}
```

响应数据: 
```json
{
  "code": "8888"
}
```
